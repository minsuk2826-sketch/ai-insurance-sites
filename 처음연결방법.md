# 처음 연결 순서

1. GitHub 로그인
2. 새 저장소 생성
   - Repository name: `ai-insurance-sites`
   - Private 권장
   - README 자동 생성은 체크하지 않음
3. 이 ZIP을 압축 해제
4. GitHub 저장소 화면에서 `uploading an existing file` 선택
5. 압축 해제한 폴더 안의 파일과 4개 폴더를 모두 업로드
6. `Commit changes` 클릭
7. Netlify에서 각 프로젝트별로 `Import from Git` 선택
8. GitHub 저장소 `ai-insurance-sites` 선택
9. 프로젝트마다 Base directory를 해당 폴더로 지정
10. Build command는 공란, Publish directory는 `.`

주의: 현재 Netlify 크레딧 제한이 걸려 있으므로, 연결 자체는 가능해도 새 Production deploy는 크레딧이 풀린 뒤 진행될 수 있습니다.
