GitHub 업로드 방법

1. 이 ZIP을 다운로드하고 압축을 풉니다.
2. GitHub의 ai-insurance-sites 저장소로 이동합니다.
3. Add file → Upload files를 누릅니다.
4. 압축을 푼 폴더 안의 3개 폴더를 모두 드래그합니다.
5. 같은 경로의 index.html이 새 파일로 교체되는 것을 확인합니다.
6. Commit changes를 누릅니다.
7. Vercel이 세 프로젝트를 자동으로 다시 배포합니다.

적용 기능
- 완료 화면 우측 상단 X 버튼
- 어두운 바깥 배경 클릭 시 닫기
- 키보드 ESC로 닫기
- 닫은 뒤 기존 랜딩페이지 화면으로 복귀
