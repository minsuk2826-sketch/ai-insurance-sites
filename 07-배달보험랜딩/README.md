# 07-배달보험랜딩 최종본

GitHub 저장소의 `07-배달보험랜딩` 폴더 안에 아래 파일을 모두 업로드하세요.

- index.html
- style.css
- script.js
- config.js
- vercel.json

## 업로드 방법

1. GitHub에서 `07-배달보험랜딩` 폴더를 엽니다.
2. `Add file` → `Upload files`를 누릅니다.
3. 이 폴더 안의 파일 5개를 모두 끌어다 놓습니다.
4. 같은 이름의 기존 파일은 새 파일로 교체됩니다.
5. `Commit changes`를 누릅니다.
6. Vercel 프로젝트의 Root Directory는 `07-배달보험랜딩`으로 설정합니다.

## 연결 완료

- 카카오 오픈채팅: https://open.kakao.com/o/sPtglPDi
- CRM 관리자 화면: https://ai-insurance-crm.vercel.app/

## CRM 실제 저장

CRM 관리자 화면 주소는 고객 데이터를 받는 API 주소와 다릅니다.
실제 저장 API 주소를 확인한 뒤 `config.js`의 `CRM_API_URL`에 입력해야 합니다.
API 주소를 비워둔 상태에서도 폼 검증과 접수 완료 화면은 작동합니다.
