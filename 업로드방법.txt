정정본 적용 방법

1. 이 ZIP 압축을 풉니다.
2. GitHub ai-insurance-sites 저장소에서 Add file → Upload files를 누릅니다.
3. 아래 두 폴더를 통째로 업로드합니다.
   - 02-리크루팅랜딩페이지1
   - 03-육아경단녀랜딩페이지
4. 기존 index.html을 교체하는 내용이 맞는지 확인합니다.
5. Commit changes를 누릅니다.
6. Vercel 자동배포가 끝난 뒤 두 페이지에서 테스트 신청합니다.

이 정정본은:
- 구글폼 연결 제거
- Supabase customers 테이블 CRM 저장
- 유입경로 각각 자동 저장
- 완료 화면 X 버튼
- 바깥 배경 클릭 닫기
- ESC 닫기
를 포함합니다.
